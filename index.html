A Map of Geometric Number Theory

About this interactive graph

The below graph is a representation of many theorems in a field of mathematics called geometric number theory (alternatively called lattice point geometry) and the mathematical connections between them. This beautiful corner of mathematics is unique in its austere simplicity and emergent complexity. Rather than doing mathematics on the whole set of real numbers (denoted ℝ), or the higher dimensional extensions of the reals (denoted ℝn), geometric number theory restricts us to those numbers which have only integer components. That is, the integer lattice, denoted ℤn. Here, most of the theorems deal only with ℤ2 -- the two dimensional integer lattice.

How to read the map

Each node in the network represents a theorem. That is, a mathematical statement pertaining to the integer lattice. These theorems rest on the firm foundation of mathematical proof, many building off one another. For example, we can use Euler's theorem for planar graphs to prove Pick's theorem so there is an arrow moving from the node for Euler's theorem to Pick's theorem. Note that not all theorems have nice names, so we have described them in as much detail is necessary when the corresponding node is highlighted using the mouse. Some theorems have a label "Ex_" which denotes a particular exercise in a particular course reader. These will likely be removed in short order and replaced with something more representative of the theorems they denote.
